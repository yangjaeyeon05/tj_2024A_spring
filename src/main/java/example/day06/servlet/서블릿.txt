HTTP : 하이퍼텍스트 전송 프로토콜
    - 하이퍼텍스트 : 이동이 가능한 텍스트/문자 , HTML , 일반 문자
    - 전송 : 통신 , 주고 받기
    - 프로토콜 : 규약 , 규칙 , 법
    - 통신 방식/메소드 : 주어진 리소스(자료)에 수행하길 원하는 행위/메소드
        - POST(보낼때/저장C) , GET(받을때/조회R) , PUT(보낼때/수정U) , DELETE(삭제할때/삭제D)
            - 관례적으로 사용되고 가독성을 위해 구분해서 사용 , 만일 GET으로 CRUD 구현한다고 하면 가능하다.

    - 통신 데이터 타입
        Content-Type : 1. text/plain 2. application/jason
    - 데이터 전송 방법
        [1] 쿼리스트링 이용 [GET,POST,PUT,DELETE 모두 사용 가능]
            - URL상의 매개변수가 노출 (보안 취약)
        [2] BODY 이용 [POST,PUT 만 사용 가능]
            - HTTP 본문에 데이터를 작성(형식 : JSON , TEXT , HTML , XML 등등)

서블릿 : 자바를 사용하여 웹페이지를 동적으로 생성하는 서버측 프로그램
        - 웹 서버의 성능을 향상하기 위해 사용되는 일종의 클래스
        - 즉) 자바 회사에서 웹 개발을 하기 위한 웹 관련 클래스 제공한다.
        - 서블릿 사용하는 방법
            1. 해당 클래스에 HttpServlet 상속받는다. String 클래스
            2. 외부로부터 해당 클래스를 연결/매핑 할 수 있도록 HTTP url(인터넷 웹 주소) 정의하기
            - 해당 클래스에 @(어노테이션)WebServlet(/"URL")
            3. HttpServlet의 다양한 메소드 제공      , 웹서버 = 톰캣 = AWS
                - init()        : (최초 서블릿 요청 시 1번) 웹서버 내 해당 클래스가 객체를 생성할 때 실행되는 메소드/함수
                - service()     : (요청이 들어올때 마다) HTTP를 이용한 웹서버로부터 요청이 들어올 때 실행되는 메소드/함수
                - destroy()     : (웹서버가 종료될 때 1번) 웹서버내 해당 클래스의 서블릿 객체가 초기화될 때 실행되는 메소드/함수 * 스프링에서 확인 불가
                - doGet()       : (HTTP GET 요청) HTTP를 이용한 웹서버로부터 요청이 들어올때 HTTP의 METHOD가 GET이면 호출되는 메소드/함수
                - doPost()      : (HTTP POST 요청) HTTP를 이용한 웹서버로부터 요청이 들어올때 HTTP의 METHOD가 POST이면 호출되는 메소드/함수
                - doPut()       : (HTTP PUT 요청) HTTP를 이용한 웹서버로부터 요청이 들어올때 HTTP의 METHOD가 PUT이면 호출되는 메소드/함수
                - doDelete()    : (HTTP DELETE 요청) HTTP를 이용한 웹서버로부터 요청이 들어올때 HTTP의 METHOD가 DELETE이면 호출되는 메소드/함수

            4. HTTP 요청의 자료/데이터 주고 받기
                http://ip주소:port번호/서블릿주소?key=value
                    - 쿼리스트링 url?key=value?key=value?key=value
                1. HttpServletRequest   : 클라이언트로부터 요청 받을 때 요청 정보를 가지는 클래스
                    - 메소드
                        .getParameter("key") : 해당하는 key의 value 데이터를 문자열 타입 반환
                2. HttpServletResponse  : 클라이언트로부터 응답 보낼 때 응답 정보를 가지는 클래스
                    - 매소드
                        .getWriter().print("value") : 해당하는 value를 요청보낸 클라이언트에게 응답 값을 보낸다.
