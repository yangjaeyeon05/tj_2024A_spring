- spring web
    @Controller
        - 스프링MVC에서 클라이언트의 요청을 처리하고 클라이언트에게 응답 처리하는 역할
        - 요청 매핑 : @Controller 클래스내 @OOOMapping 어노테이션들의 특정 URL 경로와 메소드를 매핑/연결한다.
        - 반환 종류
            1. 일반적으로 모델/값(@ResponseBody) 반환 , 주로 AJAX와 통신할 때 사용
            2. 뷰 반환(템플릿은 렌더링 후 클라이언트에게 응답) , 주로 페이지 요청할때 사용

    @RestController
        - @Controller + @ResponseBody 조합된 컨트롤러
        - 해당 컨트롤안에 있는 메소드들은 모두 @ResponseBody 타입의 반환 값을 한다.

    @RequestMapping("/클래스내공통URL")
        - 클래스 위에 사용 시 , 클래스내 모든 메소드 매핑 URL의 공통 URL 사용된다.

    @PostMapping("URL")
        - 메소드 위에 사용 , 메소드와 매핑할 HTTP의 URL 정의하고 HTTP POST METHOD 사용
